创建发起或修改一个项目在github上面
1、首先进入GitHub的首页创建一个项目，进入页面停留。
2、打开windows的cmd命令行进入在本地为这个项目创建的文件夹（注意要在git的安装文件夹下面创建）
3、进行初始化，输入命令：
git config –global user.name ‘linbaba22’
git config –global user.email ‘邮箱地址’
git config -l
到这一步就是初始化完成了。
4、下面进行远程数据库和本地的链接，输入命令：
git init
git remote add origin https://github.com/linbaba222/blockchain.git#这里的这个网址是创建的项目的数据仓库地址，在网页中会给出。
5、现在就可以在本地的项目文件夹里面增删文件了，增删文件完成之后，cmd命令行输入命令：
git status
用来检查本地文件发生了哪些变化。
下面会输出你修改或创建的文件名字。
6、检查之后没有问题之后，输入命令推送本地文件到数据仓库：
git add *
git commit -m ‘first_commit’
#到这一句仅仅在本地生效了，下面一句将文件推送到远端
git push -u origin master
7、此时刷新刚刚创建的项目停留的页面，就会显示刚刚推送到GitHub的文件。
