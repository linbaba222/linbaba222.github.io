1、 最长回文子序列
Longest Palindromic Subsequence

Given a string s, find the longest palindromic subsequence’s length in s. You may assume that the maximum length of s is 1000.

Example 1:
Input:

“bbbab”
Output:
4
One possible longest palindromic subsequence is “bbbb”.
Example 2:
Input:

“cbbd”
Output:
2
One possible longest palindromic subsequence is “bb”.

解题思路：还是动态规划的练习，这道题需要求出输入字符串的最长回文子序列，这里的子序列即指按照输入字符串的顺序对字符串进行任意切片合成得到的为子序列，根据动态规划思想，我们要初始化一个dp数组并正确得初始化这个数组来正确的从前面得状态推导出后面的状态，即状态转移方程，
首先明确这个dp数组的定义，定义一个一维数组肯定是不行的，若要用一维数组代表到字符串的前几位的最长回文子序列，那肯定找不到整个字符串最长的回文序列，那么就使用二维的dp数组来定义，dp[i][j]来表示字符串第i位到第j位切片的最长回文子序列，那么这样肯定就能遍历到这个字符串的所有子序列了，注意的是这个二维的dp数组只有一半是使用的另一半无意义，因为i小于j的时候也就无意义的。
建立状态转移方程，对dp[i][j]的推导来说，首先查看s[i]与s[j]是否相等，如果相等的话，那么对dp[i+1][j-1]表示第i位到第j位的最长子序列，那么dp[i][j]则直接等于dp[i+1][j-1]+2即可，这样加上两边的相等的字符。若s[i]与s[j]不相等的话，那这时将s的第i位字符和第j位字符分别加入字符串i+1到j-1的切片看哪个比较大即可即dp[i][j] = max(dp[i+1,j],dp[i][j-1])
这样就可以看出该怎样设置初始状态，把这个二维数组视为一个矩阵，则dp[i][j]由dp[i][j]左下的元素和左边和下面的元素推导出来，那么初始化dp数组就可以将这个矩阵的对角线全部填充为1，为了保证可以实现全部的状态转移，行逆向遍历填充，列正向遍历填充，那么这样填充到最后一个元素也就是dp[0][len(s)-1]就是最后的答案。

2、 石子游戏
StoneGAme
Alex and Lee play a game with piles of stones. There are an even number of piles arranged in a row, and each pile has a positive integer number of stones piles[i].

The objective of the game is to end with the most stones. The total number of stones is odd, so there are no ties.

Alex and Lee take turns, with Alex starting first. Each turn, a player takes the entire pile of stones from either the beginning or the end of the row. This continues until there are no more piles left, at which point the person with the most stones wins.

Assuming Alex and Lee play optimally, return True if and only if Alex wins the game.

Example 1:

Input: [5,3,4,5]
Output: true

Explanation:
Alex starts first, and can only take the first 5 or the last 5.
Say he takes the first 5, so that the row becomes [3, 4, 5].
If Lee takes 3, then the board is [4, 5], and Alex takes 5 to win with 10 points.
If Lee takes the last 5, then the board is [3, 4], and Alex takes 4 to win with 9 points.
This demonstrated that taking the first 5 was a winning move for Alex, so we return true.

Note:

2 <= piles.length <= 500
piles.length is even.
1 <= piles[i] <= 500
sum(piles) is odd.

解题思路：还是动态规划练习题，这是一道关于博弈论得题，使用动态规划得思想来解决他，两个人在分石子，题目规定一个人先手并且说明每个人都是聪明的，即每个人选的石子堆都是选的最优解，这样最后返回先手是否大于后手得石子数，是则返回true，否则返回false，根据动态规划思想，先首先初始化这个dp数组，进行明确dp数组的含义，对这道题，石子即是一个数组piles罢了，里面的每个数即是石子堆的石子数，

创建一个二维数组来对结果进行推导，dp[i][j]储存的即是对这个石子堆的数组的第i到j项的先手或后手的结果，在数组中存入元组来表示对这一切片的先手后手结果，这样初始状态就可以推导出来了，这个二维数组即矩阵的对角线填入(piles[i],0)，因为先手就直接将这个石子堆选走了，后手没得选只能是零，二维数组的其他位置先填入(0,0)，这里的二维数组同上道题相同，矩阵的左下部分是无意义的。
现在确定状态转移方程，对dp数组中的每个元素表示的数组的切片，它的选择只有两种，选择左边或者选择右边的那堆石头，那么这样状态转移就可以这样推导，如果选择左边的那堆石头，那么查看dp[i+1][j]的后手，因为此时相当于我选择了左端的石头，那么石头还剩[i+1][j]那此时我相当于后手，那么这样查看dp[i+1][j]的后手状态即可，选择右端的石头同理这样分析，比较一下两种情况那种更大，让dp[i][j]的先手状态等于那种更大的结果，那在此处的后手状态等于dp[i+1][j]因为已经选走了右端的石子堆，这时在此处的后手状态即是dp[i+1][j]的先手状态。
最后依次填充，到最后比较dp[0][n-1]的先手状态减去后手状态是否大于零即可。
