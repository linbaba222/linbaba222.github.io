最近在GitHub看labuladong大佬的学习教程项目做了如下笔记（小声bb，其实是因为我看算法书有点看不下去了）
1、动态规划框架
#dp数组的核心思想就是使用空间来换取时间，所以要构建一个dp数组首先要看数组中存储的元素能不能穷举覆盖到所有的题中的可能状态。
#认为最重要的就是尽量减少重叠的问题，来提高算法的效率。
#重要的是明确dp数组所存数据的含义，要明确它，根据dp数组的定义想办法运用数学归纳法来对dp数组进行延伸，ps：重要的是还要明确初始化条件来初始化dp数组保证得到正确的结果，具体的实例可以参考一下实例，leetcode最长递增子序列，在电脑上的dp练习也有这个的实例，
动态规划的思路一般是为自底而上对问题进行求解。
动态规划思想的三个要素：重叠子问题，最优子结构，状态转移方程，
对于状态转移方程往往是最困难的，一个思维框架：
明确「状态」 -> 定义 dp 数组/函数的含义 -> 明确「选择」-> 明确 base case。
最优子结构：最优子结构并不是动态规划问题专属的，而是某些问题的一种特定性质，通过子问题的最优值来得到规模更大的主问题的最优解，要想满足最优的子结构，必须最问题之间相互独立，才能得到最优解。
