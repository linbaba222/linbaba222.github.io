1、wiggle subsequence
A sequence of numbers is called a wiggle sequence if the differences between successive numbers strictly alternate between positive and negative. The first difference (if one exists) may be either positive or negative. A sequence with fewer than two elements is trivially a wiggle sequence.

For example, [1,7,4,9,2,5] is a wiggle sequence because the differences (6,-3,5,-7,3) are alternately positive and negative. In contrast, [1,4,7,2,5] and [1,7,4,5,5] are not wiggle sequences, the first because its first two differences are positive and the second because its last difference is zero.

Given a sequence of integers, return the length of the longest subsequence that is a wiggle sequence. A subsequence is obtained by deleting some number of elements (eventually, also zero) from the original sequence, leaving the remaining elements in their original order.

Example 1:

Input: [1,7,4,9,2,5]
Output: 6
Explanation: The entire sequence is a wiggle sequence.
Example 2:

Input: [1,17,5,10,13,15,10,5,16,8]
Output: 7
Explanation: There are several subsequences that achieve this length. One is [1,17,10,13,10,16,8].
Example 3:

Input: [1,2,3,4,5,6,7,8,9]
Output: 2

解题思路：对于这道题首先想到的就是用对给出的整个整数序列切片，找到每一个上升下降交替的小子序列，加起来就是最后的最长的子序列，因为题目中给出了可以随意删除一些元素来找到那个最大的子序列，可以使用回溯算法：
Python：
class Solution:
def calcuate(self,nums,index,isup):
maxcount = 0
for i in range(index+1,len(nums)):
if (((not isup) and nums[i])>nums[index]) or (((isup) and nums[i])<nums[index]):#判断条件是上升序列还是下降的，相当于将符合条件的每一个小子序列全部都切出来
maxcount = max(maxcount,1+self.calcuate(nums,i,not isup))#递归回溯，找到上升下降交替的序列
return maxcount
def wiggleMaxLength(self, nums: List[int]) -> int:
if (len(nums)<2):
return len(nums)
return 1+max(self.calcuate(nums,0,True),self.calcuate(nums,0,False))#判断从头开始上升还是下降的序列更多

第二个思路使用动态规划思想，将给定的整个序列想象成一个个波峰与波谷，在每一个序列值处，索引看作横坐标轴，序列值看作为竖坐标值，求解最大摆动序列的值，可以看作，求这张图里面的波峰波谷数量为多少，使用两个数组来保存状态，一个数组用来保存上升的序列值，也就是波峰的数量，而且在添加这个数组时，必须要满足此时的横坐标值要大于另外一个数组的波谷数量，而且此时的序列值要大于前一个波谷值，另一个数组同样。
Python：
class Solution:
def wiggleMaxLength(self, nums: List[int]) -> int:
if (len(nums)<2) :
return len(nums)
up = []
down = []
for i in range(len(nums)):#初始化数组
up.append(0)
for i in range(len(nums)):
down.append(0)
for i in range(len(nums)):
for j in range(i):
if nums[i]>nums[j]:
up[i] = max(up[i],down[j]+1)#增长递增数列的长度
elif nums[i] <nums[j]:
down[i] = max(down[i],up[j]+1)#增长递减数列的长度
return 1+ max(down[len(nums)-1],up[len(nums)-1])

斐波那契数列：
Garlic King likes to climb stairs very much, this time, he obtains a special power, each time can jump any odd number of stairs. For example, he was initially at the bottom of the building, crossing-stairs to ladder 1, or across three stairs to ladder 3. As shown below

In order to choose one of the easiest ways to climb the stairs, garlic wants to try all the different ways to get to the top of the building. For-there are n steps of stairs, garlic King-how many general ways to get to the top of the building from the bottom of the building.

Since the final answer may be large, output the final answer to the result of the 100007 model.

Input format

The first line enters an integer n (1 ≤ n ≤ 1000).

Output format

Output the result of the final answer for 100007 of the model.

Sample input

4

Out

3

解题思路：额，，，对你没看错，这就是那个蒜头爬楼梯的，就是斐波那契数列，嗯，，，用python有许多各式各样的优化求解速度的代码，现在在这里整理一下这些方法，拓展一下思路。
第一种是递归，因为重复求值比较多，所以递归求解会比较慢：
Python代码：
def fib(n):
if n<=1:
return n
else:
return fib(n-1)+fib(n-2)#递归

def main(n):
if n>0:
print((fib(n))%100007)
else:
print(‘error!’)

n = int(input(‘输入n：’))
main(n)

第三种是使用python的生成器语法：
def loop(n):
a,b = 0,1
while n>0:
a,b = b,a+b
n-=1
yield a#每次都会向生成器外返回本次的斐波那契数列第n项

n = int(input(‘输入n：’))
for i in loop(n):
print(i)

第五种方法：使用python自带的numpy科学计算库，实现矩阵快速幂计算斐波那契数列
优化斐波那契数列的递推式，实际上就是一个相同的矩阵“1 1：1 0”与斐波那契数列的第0项与第一项相乘，具体可以使用一张图来解释

Python代码：
def tool(n):
mat = np.matrix(“1 1;1 0”,dtype=’int64’)#创建矩阵”1 1:1 0”
return np.linalg.matrix_power(mat,n)#使用numpy内置方法计算矩阵幂

def matrix(n):
result = []
for i in range(n):
result.append(np.array(tool(i))[0][0])
return result

n = int(input(“输入n：”))
matrix(n)
