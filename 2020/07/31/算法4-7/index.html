1、Sorted Colors
Given an array with n objects colored red, white or blue, sort them in-place so that objects of the same color are adjacent, with the colors in the order red, white and blue.

Here, we will use the integers 0, 1, and 2 to represent the color red, white, and blue respectively.

Note: You are not suppose to use the library’s sort function for this problem.

Example:

Input: [2,0,2,1,1,0]
Output: [0,0,1,1,2,2]
Follow up:

A rather straight forward solution is a two-pass algorithm using counting sort.
First, iterate the array counting number of 0’s, 1’s, and 2’s, then overwrite array with total number of 0’s, then 1’s and followed by 2’s.
Could you come up with a one-pass algorithm using only constant space?

解题思路：这道题是一个很简单的排序的问题，可以新建出一个新数组进行排列，另外的进阶算法是要求使用一次遍历就原地将数组排序，因为这道题的特殊性，可以使用三个指针进行排序，首先两个头尾双指针初始化，然后一个查看当前指针状态的指针，这三个指针就相当于分别代表0，1，2三个数字，保证这个指针的左边都是排序好的01数列，当当前指针是2的时候，就将当前指针和尾部指针交换，并将尾部指针向前缩小，而此时当前状态指针由于不能保证交换过来的尾部指针是什么，所以当前位置指针不可以向后移动，当当前指针是0的时候就将当前指针和头部指针交换值，可以保证数列头部都是0，随后当前指针与头部指针都右移，这样就可以一次遍历实现这个三色数组的排序。

另外因为最近在学习排序算法，然后将学到的快速排序算法在这道题里面应用了一下，快速排序算法是一种递归的方式将数组分成了几个不同的部分再进行，快速排序算法首先在数组首部设定一个key值来进行头尾双指针遍历数组，将小于key值的数扔到数组的左边（这里是从小到大排序），然后将大于key值的数扔到数组的右边，最后到双指针相等，再进行对key值得重新赋值，再次调用自身函数递归，将数组按照双指针相等的位置，再次分为两个部分重新进行遍历，重复进行，最后即可得到原地修改排序后的排序数组，是一种对冒泡排序算法的改进。

2、perfect squares
Given a positive integer n, find the least number of perfect square numbers (for example, 1, 4, 9, 16, …) which sum to n.

Example 1:

Input: n = 12
Output: 3
Explanation: 12 = 4 + 4 + 4.
Example 2:

Input: n = 13
Output: 2
Explanation: 13 = 4 + 9.

解题思路：可以使用bfs创造一个队列，再创建一个集合目的是为是集合内元素无重值，也就是避免重复的对同一个数值进行操作，大大减少了时间复杂度，初始化在队列里面添加一个元素，一个元素包括上一个数减去一上一个平方数的余值，和减去的平方数的数量，每次对队列中弹出第一个元素，对第一个元素进行操作，每次操作创造一个列表包含使用这个余值减去一些平方数的数值，再对这个列表进行遍历，如果有等于零的数则直接输出队列元素的平方数数量加一，若这个数没有在集合之中，那么在继续在队列右侧（因为前面的还没有检测完，我们需要的是最小的平方数之和）添加元素，余值依旧，而步骤，也就是平方数的数量加一，同时集合中也要加上这个余值。返回的步骤数，也就是平方数之和，就是最后的结果。
